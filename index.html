CrowPi Assistant

CrowPi Assistant is an AI-powered learning companion designed specifically for the CrowPi educational computer kit.

Key features:

  • Voice command recognition
  • Integration with OpenAI API for intelligent responses
  • Context-aware help for CrowPi components and programming
  • Real-time circuit visualization when running on CrowPi hardware
  • Flask backend with JavaScript frontend for responsive UI
  • Spanish is his principal language

Example commands:

  • "Show me how to blink an LED"
  • "Explain GPIO pins"
  • "Give me a Python example for the temperature sensor"

Contapp

Contapp was initially created by students in the 9th period, but it had not been completed. A friend and I decided to finish the website.

We had to finalize the web application and continue developing the mobile app. We worked on connecting the APIs with the frontend and redesigned some parts of the application.

The main scripts we developed were written in TypeScript using React for the web app, and React Native for the mobile app.

Additionally, we had to fix several backend errors and create new scripts.

The key features we implemented included creating posts, designing quizzes accessible through a teacher account, registering teachers via the admin account, forming groups with the admin account, registering students with email confirmation, and sending all data to the database.

For the mobile app, we implemented methods to open PDFs, answer quizzes, and provide students the ability to view their quiz grades.

This project was requested by a teacher, Miriam Olvera Cueyar. Her email is miriam.cueyar@upt.edu.mx.
